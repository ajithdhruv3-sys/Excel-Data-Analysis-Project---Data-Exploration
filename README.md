# Excel-Data-Analysis-Project---Data-Exploration
📊 Excel Data Analysis Project

📌 Overview
This project showcases fundamental Excel skills used in data analysis. It includes calculations, logical operations, conditional aggregations, and text manipulation techniques applied to a product dataset.

🧾** Dataset Details**
-Total Products: **34**
-Dataset includes:
-Product ID
-Category
-Price

🔢 Key Analysis Performed
 Sum, Count, Average
- Total Price of all products: **10100**
- Total number of products: **34**
- Average price: **297.06**

2️⃣ Minimum & Maximum
- Minimum Price: **30**
- Maximum Price: **1000**

3️⃣ IF Function (Price Categorization)
Created a new column **Price Range**:
Price ≥ 500 → **High Price**
Price < 500 → **Standard Price**

---

4️⃣ SUMIF & COUNTIF
Total price of **Electronics** category: **8050**
Count of products priced below 100: **11**

---

5️⃣ Text Functions (Data Transformation)

Created new columns from **Product ID**:
**Day** → First 2 characters (using `LEFT`)
**Month** → 4th to 6th characters (using `MID`)
**Country Code** → Last 2 characters (using `RIGHT`)

---

🛠️ Skills Demonstrated
- Data aggregation using SUM, COUNT, AVERAGE
- Logical operations with IF
- Conditional calculations using SUMIF & COUNTIF
- Text manipulation with LEFT, RIGHT, MID
- Data cleaning and transformation
